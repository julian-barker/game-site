How-To-Play

Control Keys

  • Left arrow key - Move game piece left

  • Right arroww key - Move game piece right

  • Up arrow key - Rotate game piece clockwise

  • Down arrow key - Hard drop game piece

  • Spacebar Key - Pause game

Scoring

Scoring is achieved by clearing rows of the playing field. When a row is completly filled with at least one cube of a game piece the row will clear. You will recieve (100)points per row cleared.
